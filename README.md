# 夜归小站 Night Station

《夜归小站》是一款轻量 2D 治愈经营小游戏。

玩家经营一间只在夜晚营业的小站，接待疲惫、失眠、焦虑、沉默的夜归人，通过饮品、灯光、座位和小物件缓解访客情绪，解锁短故事，并获得“微光”资源，用来修复和装饰小站。

## 游戏定位

- 类型：2D 治愈经营 / 情绪修复 / 轻量互动
- 平台：Web / H5 / 微信小游戏方向
- 节奏：单局 5–10 分钟
- 操作：点击选择
- 风格：夜晚、暖灯、雨声、小店、低压力、微叙事

## 核心循环

访客到来 → 玩家观察访客状态 → 选择饮品 / 灯光 / 座位 / 小物件 → 访客情绪变化 → 解锁短故事 → 获得微光 → 修复或装饰小站 → 迎接下一位夜归人

## 第一阶段 MVP

- 一个夜晚小站主界面
- 一个访客角色
- 一个情绪值
- 三个饮品按钮：热茶、咖啡、糖水
- 一个反馈文本区域
- 一个故事解锁机制
- 一个“微光”资源
- 一个“重新开始一天”按钮

## 第一阶段玩法规则

初始情绪值：30

点击“热茶”：情绪 +15  
点击“咖啡”：情绪 +5  
点击“糖水”：情绪 +10  

情绪达到 60：解锁第一段访客故事  
情绪达到 90：访客被安抚，玩家获得 1 点“微光”  

点击“重新开始一天”：重置情绪、故事、微光状态
