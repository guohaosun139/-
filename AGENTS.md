# AGENTS.md

## 项目说明
《夜归小站 Night Station》是一个使用 **HTML + CSS + Vanilla JavaScript** 构建的轻量原型项目。

## 约束
- 不引入任何前端框架。
- 不引入复杂依赖。
- 以可读性和可扩展性优先，保持模块化结构。
- 当前阶段使用文字、色块和简单几何图形作为视觉占位。

## 代码结构建议
- `index.html`：页面结构与基础样式容器。
- `src/main.js`：应用入口，绑定 UI 与交互。
- `src/game.js`：核心玩法状态与规则逻辑。
- `src/data.js`：阶段常量、文案和配置。

## 开发建议
- 单一职责：UI 渲染、状态计算、配置数据分离。
- 所有阈值与数值尽量配置化，避免硬编码散落。
- 每次迭代优先保证“可运行、可验证”。
